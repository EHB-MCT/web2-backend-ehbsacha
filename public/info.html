API documentation

Root route

  • / = root info page

User routes

  • GET /users = get all users
  • GET /user = login by sending required: name, password
  • POST /user = Create a new user required: name, password, email
  • PUT /user/name = Change userName required: name, password, newName
  • PUT /user/password = Change password required: name, password, newPassword
  • DELETE /user = Delete a user and all liked and shelved games required: name, password

Like routes

  • GET /likes = Get all liked games of user required: userId, gameId
  • POST /like = Create a new liked game required: userId, gameId
  • PUT /like = Change liked status required: userId, gameId

Shelf routes

  • GET /shelved = Get all shelved games of user required: userId, gameId
  • POST /shelf = Create a new shelved game required: userId, gameId
  • PUT /shelf = Change shelved status required: userId, gameId

Like and shelf common routes

  • GET /game = Check if game exists for user in likes and shelf
  • DELETE /game = Delete a game for user
